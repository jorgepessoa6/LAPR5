Para criar uma conta na OPT – Otimização e Planeamento de Transportes, é necessário concordar com as nossas Políticas de Privacidade.

Quando cria uma conta, processamos os seus dados pessoais, tendo em conta os seguintes pontos:

  • Armazenamos as informações que nos fornece, como nome e endereço de e-mail, a data de nascimento e a localização do cliente para controlo de serviços.
  • Vamos guardar o seu nome e e-mail para termos forma de o identificar na nossa base de dados.
  • Utilizamos o endereço de email para entra em contacto com o cliente. Esta comunicação tem como objetivo de tratamento dos dados do cliente e mudanças na nossa Política de Privacidade.
  • O cliente deverá ter a idade minima de 13 anos para dar consentimento.
  • A localização do cliente nunca é guardada armazenada podendo este desativar a partilha da mesma no seu diapositivo. Apenas é necessária no momento de utilização do componente de visualização, o Mapa da Rede, para que o cliente se possa situar com precisão no mapa.
  • No caso de inatividade por parte do cliente, os dados permaneceram no sistema durante o período de um ano. No fim deste período os dados serão removidos do sistema.

Em qualquer altura, é possível ao cliente, aceder aos seus dados (podendo mesmo alterá-los), a forma de tratamento dos mesmos e a sua finalidade.

O cliente pode também solicitar a eliminação de certos dados:

  • Se estes deixam de ser necessários para a sua finalidade.
  • Quando é retirado o consentimento sobre os mesmos e não existe outro fundamento jurídico para o mesmo.
  • Quando se oponha ao tratamento e não existam interesses legítimos prevalecentes.
  • Quando os dados pessoais foram tratados ilicitamente.
  • Quando os dados pessoais têm de ser apagados para o cumprimento de uma obrigação jurídica.
  • Quando os dados pessoais foram recolhidos no contexto da oferta de serviços da sociedade da informação.
